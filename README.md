# Factory Method (Java)

Этот проект я сделала по задаче, где нужно создавать объекты разных типов по входным данным.
Я выбрала паттерн Factory Method, потому что он позволяет создавать объекты через один метод
и не писать конкретные классы напрямую в основном коде.

В проекте есть общий интерфейс Product и класс ProductFactory, где находится метод createProduct(type).
Он принимает тип (например, A или B) и возвращает нужный объект. Если тип не поддерживается,
то выводится ошибка.
